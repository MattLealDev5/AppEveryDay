# App Every Day

A challenge to build one iOS app per day using Xcode Intelligence alongside Claude Code, with the goal of better understanding how AI works and how to leverage it effectively during development.

## Projects

### Day 1 — MicroAdventures / MyAdventures
Two nearly identical apps that randomly generate a "micro adventure" near your location.
- Video: [Xcode Intelligence Live Stream](https://www.youtube.com/live/U1WM3ALwQX8?si=CdkOb7cx-J0PEc0_)

### Day 2 — HabitTracker
A todo-style app designed to help build and maintain daily habits.
- Video: [HabitTracker Tutorial](https://youtu.be/QdpaI_j0FRU?si=XOGAgjEQzS0OaI5W)

### Day 3 — PersonalFinance
An app for tracking financial spending and staying within budgets.
- Video: [PersonalFinance Tutorial](https://youtu.be/QdpaI_j0FRU?si=XOGAgjEQzS0OaI5W)

### Day 4 — BipTheGuy
A novelty app where you upload an image and "bip" it. This was the first project where prompts were written independently, applying lessons learned about effective AI prompting.
- Video: [BipTheGuy Tutorial](https://youtu.be/ouPcMasRDhY?si=eJE3YPGl5Xaeyye5)

### Day 5 — CoffeeDelivery
A more ambitious frontend app for a coffee delivery service, built using a Figma community design file alongside Figma's dev mode to explore Apple's independent AI tools.
- Design: [Figma Community File](https://www.figma.com/community/file/1245817631079079109)
